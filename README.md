# mock-issuer

## overview
ホワイトラベルウォレット開発用のissuerとveriferのmock

## Getting Started

### ngrokの用意
ngrokを使うにはアカウントの登録が必要になります
ngrokの使い方ガイド
https://kakechimaru.com/ngrok/


First, run the ngrok server:
```bash
ngrok http [port番号]
```
First, run the development server:

```bash
npm run dev
# or
yarn dev
```
